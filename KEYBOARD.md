# Keyboard shortcuts

| Key | Action |
| --- | --- |
| ← ↑ → ↓ | Move focus |
| Enter | Activate focused control |
| Escape | Back / close |
| Tab / Shift+Tab | Next / previous focus |
| S | Focus search |
| ↑ / ↓ (reader) | Previous / next verse |
| ← ↓ (resizer) | Narrower panel |
| → ↑ (resizer) | Wider panel |
| Home / End (resizer) | Min / max width |
| Enter (add verse) | Add verse |
| Enter / Escape (confirm) | Confirm / cancel |
| Ctrl+Shift+R | Reset reading position |
| Ctrl+Shift+B | Change Bible version |
